# Final Report Structure

Use this structure after clarification, research, synthesis, and skeptic loops.

```text
Idea:
  one paragraph

Verdict:
  worth doing / worth modifying / not worth doing yet

Why:
  evidence-backed reasons

Modified Idea:
  improved version after research and critique

MVP:
  smallest useful version

Implementation Path:
  phases, key components, dependencies

Validation Experiments:
  what to test before building fully

Major Risks:
  ranked risks and mitigations

Open Questions:
  remaining unknowns

Next Step:
  one concrete action
```
